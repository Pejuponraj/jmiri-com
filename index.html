6 Types of Correlation in Data Science

Below is a summary of commonly used correlation types in IT and data science:

Type Formula (Key Variables) When to Use Example (IT)
Pearson r = ∑(X−X̄)(Y−Ȳ) / sqrt(∑(X−X̄)² ∑(Y−Ȳ)²) Both variables continuous; linear relationship Experience vs Salary
Spearman ρ = 1 - (6∑d²)/(n(n²-1)), where d = rank diff Ordinal/ranked data, monotonic relation Rating vs Product Returns
Kendall Tau τ = (C - D) / (n(n-1)/2) Small/tied ranked data Search result vs user clicks
Point-Biserial r = (Mean1 - Mean0) / StdDev × sqrt(n1n0/n²) Binary + Continuous Clicked vs Time Spent
Phi Coefficient ϕ = (AD - BC) / sqrt((A+B)(C+D)(A+C)(B+D)) Two binary variables Fraud vs Foreign Txn
Partial rXY.Z = (rXY - rXZrYZ) / sqrt((1−rXZ²)(1−rYZ²)) Remove third variable effect Study Hours vs Marks (controlling Sleep)
