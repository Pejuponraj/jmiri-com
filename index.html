6 Types of Correlation in Data Science

1. Pearson Correlation

Variables: X and Y - continuous variables

Use When: You want to check linear relationship between numeric variables.

Example: Experience vs Salary in IT field

2. Spearman Rank Correlation

Variables: d - difference between ranks of X and Y

Use When: Data is ordinal or ranked, and relationship is monotonic.

Example: Product rating rank vs number of returns

3. Kendall Tau Correlation

Variables: C - Concordant pairs, D - Discordant pairs

Use When: Small sample size with tied ranks.

Example: User click rank vs search result rank

4. Point-Biserial Correlation

Variables: M1 & M0 - means for binary groups, SD - std dev

Use When: One variable is binary, the other is continuous.

Example: Clicked Ad (yes/no) vs Time Spent on site

5. Phi Coefficient

Variables: A, B, C, D - values from 2x2 contingency table

Use When: Both variables are binary.

Example: Fraudulent Transaction vs Foreign Location (yes/no)

6. Partial Correlation

Variables: r values are Pearson correlation coefficients

Use When: You want to control the effect of a third variable.

Example: Study Hours vs Marks (controlling for Sleep)
