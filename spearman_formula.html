Spearman Rank Correlation

Measuring the strength and direction of a monotonic relationship between ranked variables

What is Spearman Rank Correlation?

The Spearman Rank Correlation coefficient (ρ or rs) measures the strength and direction of the monotonic relationship between two variables based on their ranks. It is a non-parametric measure and useful when data are ordinal or not normally distributed.

Range: -1 (perfect negative) to +1 (perfect positive)

Real-Time Project Question

Question: Analyze the correlation between employee performance ranks and customer satisfaction ranks to help improve appraisal and reward systems in a company.

Calculation Example

Given the following data of employee ranks for performance (X) and customer satisfaction (Y):

EmployeePerformance Rank (X)Customer Satisfaction Rank (Y)d = X - Yd²
112-11

Calculate sum of d²:

Σd² = 1 + 1 + 4 + 1 + 1 = 8

Number of pairs (n) = 5

Formula for Spearman rank correlation coefficient:

ρ = 1 - (6 × Σd²) / (n × (n² - 1))

Substituting values:

ρ = 1 - (6 × 8) / (5 × (25 - 1)) = 1 - 48 / 120 = 1 - 0.4 = 0.6

This indicates a moderate positive correlation between employee performance and customer satisfaction ranks.

Python Code with Graph

import numpy as np
import matplotlib.pyplot as plt
from scipy.stats import spearmanr

# Sample data
performance_rank = [1, 2, 3, 4, 5]
customer_satisfaction_rank = [2, 3, 1, 5, 4]

# Calculate Spearman rank correlation
correlation, p_value = spearmanr(performance_rank, customer_satisfaction_rank)

print("Spearman Rank Correlation Coefficient:", correlation)
print("P-value:", p_value)

# Plot ranks
plt.scatter(performance_rank, customer_satisfaction_rank, color='green', label='Data Points')
plt.plot(performance_rank, np.poly1d(np.polyfit(performance_rank, customer_satisfaction_rank, 1))(performance_rank), color='red', label='Trend Line')
plt.title('Employee Performance vs Customer Satisfaction Ranks')
plt.xlabel('Performance Rank')
plt.ylabel('Customer Satisfaction Rank')
plt.legend()
plt.grid(True)
plt.show()
